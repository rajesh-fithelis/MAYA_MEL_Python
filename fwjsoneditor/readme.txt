Description:
----------------
Copy the file to any location
Run the file
Automatically in the background - current working directory the config.json file will be created (write to the disk)
Automatically along with the config.json the data.json will be initialized and created (write to the disk)
Now able to add, edit and delete along with preview of string manipulations
In the "Settings"  TAB the new data file path can be changed
If data file path changed then the new_file_name.json will be created to the modified location
new path will be affected to the configuration file and then the data will be loaded from the new file
